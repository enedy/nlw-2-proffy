# NLW 2 - Proffy
Proffy é a plataforma que conecta alunos e professores de forma fácil e prática, desenvolvida durante o NLW (Next Level Weekend 2)

## Projeto Api

```bash
# Acesse a pasta server
$ cd nlw-2-proffy/server

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn start
```

## Projeto WEB

```bash
# Acesse a pasta web
$ cd nlw-2-proffy/web

$ yarn install

# Execute a aplicação
$ yarn start
```
